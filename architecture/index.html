The Architecture for mobile Applications

As I started doing serious software development, the architecture, design patters and how to structure the code was the most important things for me. As a mobile, tv, desktop application engineer for ten years, I read and worked with such things as MVVM, MVC, MVI, Flow, Composable, VIPER and others. I felt all the pain of making it wrong. Each time something was feeling wrong, the question "How can I simplify this code?" drive me to make some refactoring and try new things. Sometimes the new ideas failed, other times it shined.
